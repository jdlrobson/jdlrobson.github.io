mw.html

Utility functions for aiding the construction of HTML.

HTML construction helper functions

Source:

Example

var Html, output;

    Html = mw.html;
    output = Html.element( 'div', {}, new Html.Raw(
        Html.element( 'img', { src: '<' } )
    ) );
    mw.log( output ); // <div><img src="&lt;"/></div>
